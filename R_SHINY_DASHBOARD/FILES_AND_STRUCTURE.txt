TROUBLESHOOTING QUICK GUIDE
================================================================================

PROBLEM: "object 'db_con' not found"
SOLUTION: Restart R session, verify .env file

PROBLEM: "Cannot find shapefile"
SOLUTION: Ensure files in R_SHINY_DASHBOARD/data/

PROBLEM: Database connection refused
SOLUTION: Check PostgreSQL running (psql -h localhost)

PROBLEM: Charts not rendering
SOLUTION: Clear browser cache, reinstall plotly package

PROBLEM: Memory error
SOLUTION: Filter data range (e.g., last 2 years instead of 6 years)

See README.md for detailed troubleshooting.
