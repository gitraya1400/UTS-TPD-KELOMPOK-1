READY FOR USE
================================================================================

IMMEDIATE USE:
  1. Configure .env with database credentials
  2. Place ADMINISTRAS_PROVINSI.shp in data/ directory
  3. Run: Rscript install_packages.R
  4. Run: Rscript -e "shiny::runApp()"
  5. Visit: http://localhost:3838
